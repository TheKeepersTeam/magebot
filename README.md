# magebot
MageBot: Magical duel game inspired by Harry Potter. Battle AI with spells (Attack: Ignis/Glacies/Fulmen; Healing: Vitalis/Vitae/Sanare; Resistance: Fortitudo/Praesidium/Tutela) managing HP, Resistance, Mana. Advanced Minimax AI avoids wasteful healing. Console version: Run python MageBot/console_version/magebotcli.py, type start_dual. V1.1.4.
